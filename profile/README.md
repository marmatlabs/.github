## How we work

Our engineering principles and standards are public, in
**[marmatlabs/engineering](https://github.com/marmatlabs/engineering)**:

- **Simplicity before scale** — build for the load we have, with a path to the load we might get.
- **Secure defaults** — the safe path is the default path.
- **Infrastructure as code** — if it was not committed, it does not exist.
- **Portability over lock-in** — standard protocols, containers, no design we cannot move.
- **Cost-conscious engineering** — frugal about hardware, never about security.
